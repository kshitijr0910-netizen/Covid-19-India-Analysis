# 📊 COVID-19 Dashboard (Microsoft Excel)

## Overview

This project is an interactive **COVID-19 Dashboard** built in **Microsoft Excel**. It provides a visual representation of COVID-19 statistics across Indian states using charts, KPIs, and dynamic dashboards to help users quickly analyze pandemic trends.

## Features

* 📈 Interactive dashboard with key performance indicators (KPIs)
* 🦠 Total COVID-19 cases by state
* 💚 Recovery statistics
* ❤️ Active cases analysis
* ⚫ Death statistics
* 🗺️ State-wise comparison
* 🚦 Zone-wise classification
* 📊 Charts and visualizations for easy data interpretation

## Dashboard Includes

* **Total Cases**
* **Recovered Cases**
* **Active Cases**
* **Deaths**
* **Highest & Lowest Case States**
* **State and Zone Analysis**
* **Recovery Index**
* **Death Index**

## Tools Used

* Microsoft Excel
* Pivot Tables
* Pivot Charts
* Slicers
* Conditional Formatting
* Excel Formulas

## Project Objective

The objective of this project is to present COVID-19 data in an easy-to-understand dashboard that helps users analyze trends, compare state-wise statistics, and gain meaningful insights through data visualization.
